Javascript Object Deep Copy

Javascript objects are passed around by reference - if you really want a copy to modify, without affecting the original, you need to do get a deep copy. Unfortunately, most frameworks (including the popular "underscore"), offer a function named "clone", which is only a shallow copy.

I found and modified a simple (though not necessarily efficient) deep copy function, and present it here. It basically converts the object to JSON, then back to an object, so it doesn't work on complex objects (for example, functions are not properly handled). If you need a nested data structure copied, though, this does the trick pretty well.
